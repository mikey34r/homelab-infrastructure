# Homelab Infrastructure

## Overview
Personal home lab environment used to build and test systems, networking configurations, and cybersecurity tools.

## Objectives
- Develop hands-on experience with system administration and networking
- Simulate real-world environments for testing and learning
- Support hosting of services such as game servers and virtual machines

## Components
- Ubuntu-based servers
- Virtual machines and test environments
- Remote access via Tailscale VPN
- Network configuration (port forwarding, internal routing)

## Use Cases
- Game server hosting (Minecraft)
- System testing and troubleshooting
- Network configuration and remote access setup
- Cybersecurity lab experimentation

## Technologies
- Linux (Ubuntu)
- Virtualization platforms
- VPN (Tailscale)
- Basic networking and firewall configuration

## Status
Actively expanding with additional services and documentation
